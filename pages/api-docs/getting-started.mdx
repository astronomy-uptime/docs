Our API is HTTPS/REST for general operations such as sending `POST` requests and receiving `GET` requests.

#### Base Url

[api.astrouptime.com](https://api.astrouptime.com)

## Authentication

To access our API you need to authorize yourself, this can be done by using your user api key. Your token can be obtained from your settings page settings [astrouptime.com/panel/settings](https://astrouptime.com/panel/settings?selected=other).

Authentication is performed with the `Authorization` HTTP header as such

```json:title=HTTP%20Headers
{
  "Authorization": "api-token-here"
}
```
